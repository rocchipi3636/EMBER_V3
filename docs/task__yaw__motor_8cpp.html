void yaw_motor ( void *  p_params)

Task yaw_motor controls the power delivered to the yaw motor.

This task controls the motor that controls the yaw of the entire assembly. When no fire is detected, the task spins the assembly counterclockwise slowly in order to scan the area for a fire. When a fire is detected, the task adjusts the yaw of the assembly until it is centered on the fire. When the fire is out, the task resumes spinning slowly
